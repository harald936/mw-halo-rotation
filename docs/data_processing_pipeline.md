# Data Processing Pipeline — From Raw Catalogs to 317 Data Points

## Overview

This document describes every step from raw published catalogs to the final
317 data points used in the Omega_p analysis. Every cut, binning decision,
and derived product is documented with its source and justification.

The current live provenance status is tracked by:
```bash
conda run -n mwgd1 python scripts/17_audit_final_run_inputs.py
```

Most derived products can be regenerated by running:
```bash
conda run -n mwgd1 python scripts/14_rebuild_derived_tracks.py
```

## GD-1 (144 data points)

### Source catalog
- **Paper:** Tavangar & Price-Whelan 2025, ApJ 988
- **Data:** Zenodo 15428120 (`gd1_tbl_with_memb_prob.mrt`)
- **Contents:** 466,571 Gaia DR3 sources with membership probabilities

### Membership selection
- Membership probability cut: p > 0.5 → **2,276 members**

### Quality cuts (script: `00_clean_gd1_catalog.py`)
1. **Parallax foreground rejection:** plx > 0.3 mas AND plx/e_plx > 3 → removed 136 stars
2. **Proper motion outliers:** > 3σ from polynomial fit to PM track → removed ~30 stars
3. **Radial velocity outliers:** > 3σ from polynomial fit to RV track → removed ~25 stars
4. **Metallicity:** [Fe/H] > −1.5 removed (non-stream contamination) → removed 6 stars
- **Result: 2,079 cleaned members** saved to `data/gd1/gd1_members_cleaned.csv`

### Sky + PM track (script: `01_bin_gd1_track.py`)
- Binned into **44 bins** along phi1
- Gap/spur region (phi1 ∈ [−45, −28]) masked
- Weighted by membership probability
- Errors: 10,000 bootstrap iterations with 2.5σ iterative clipping
- Channels: phi2_med, phi2_err, pm1_med, pm1_err, pm2_med, pm2_err
- Saved to `data/gd1/gd1_track.csv`

### RV track (script: `14_rebuild_derived_tracks.py`)
- **356 members** with measured RVs (DESI DR1 + SDSS + LAMOST)
- Binned into **12 bins** along phi1 (minimum 5 stars per bin)
- Weighted mean: rv_med = Σ(rv_i / e_rv_i²) / Σ(1 / e_rv_i²)
- Error: rv_err = 1 / √Σ(1 / e_rv_i²)
- Saved to `data/gd1/gd1_track_rv.csv`

### Distance
- **Not available.** Gaia parallax SNR = 0.38 at d ~ 10 kpc (noise-dominated).
  No published photometric distance track with validated errors exists.

### Total: 44 phi2 + 44 pm1 + 44 pm2 + 12 rv = **144 data points**

## Pal 5 (31 data points)

### Source catalog
- **Paper:** Kuzma et al. 2022, MNRAS 512, 315
- **Contents:** 109 spectroscopically confirmed members of Pal 5 cluster + tails

### Track (script: `05_bin_pal5_track.py`)
- Transformed to Pal5PriceWhelan18 stream frame
- Binned into **7 bins** along phi1
- Channels: phi2, pm1, pm2, rv (all 4 from same member catalog)
- Errors: std/√n per bin
- Saved to `data/pal5/pal5_track.csv`

### Distance track (script: `14_rebuild_derived_tracks.py`)
- **Paper:** Price-Whelan et al. 2019, AJ 158, 223 (Table 2)
- **Method:** 27 RR Lyrae identified in Pal 5 region using Gaia DR2 + PS1
  - 25 with membership probability > 0.8
  - 15 in stream tails (|phi1| > 1°), 10 in cluster core
- **Positions:** Gaia DR3 cross-match of published source_ids
- **Distances:** Published RR Lyrae standard candle values (~3% precision)
- **Binning:** 15 stream RR Lyrae into **3 bins** (weighted mean, 1/σ² weights)
- **Validation:** distance gradient physically correct
  (leading tail at 19.5 kpc, cluster at 20.6 kpc, trailing tail at 22.2 kpc)
- Saved to `data/pal5/pal5_dist_track.csv`

### Total: 7 phi2 + 7 pm1 + 7 pm2 + 7 rv + 3 dist = **31 data points**

## Jhelum (24 data points)

### Source catalog
- **Paper:** Li et al. 2022 (S5 DR1)
- **Data:** `data/external/s5_pdr1_light.fits`
- **Contents:** S5 spectroscopic targets in Jhelum field

### Membership selection (script: `07_add_jhelum_orphan.py`)
- S5 Jhelum field stars with stream priority (7–9)
- Kinematic cuts: |phi2| < 2°, rv_err < 10 km/s
- **Result: ~250 members** (narrow + broad components combined)

### Sky + RV track
- Binned into **6 bins** along phi1
- Channels: phi2_med, phi2_err, rv_med, rv_err
- Errors: std/√n per bin
- Note: both narrow and broad components included (not separated)

### PM track (script: `14_rebuild_derived_tracks.py`)
- S5 catalog includes `gaia_source_id` for every member
- Queried Gaia DR3 via astroquery: retrieved PMs for **2,000 / 2,849 members**
- Transformed to JhelumBonaca19 stream frame
- Converted pm_phi1_cosphi2 → pm_phi1
- Binned into 6 bins, interpolated at track phi1 positions
- Channels: pm1_med, pm1_err, pm2_med, pm2_err
- Errors: std/√n of Gaia PM measurements per bin
- Saved to `data/jhelum/jhelum_track.csv` (PM columns added)

### Distance
- **Not available.** No RR Lyrae catalog exists for Jhelum.
  Parallax useless at d ~ 12 kpc. Single-value distance (12.4 kpc)
  from Awad+2023 but no gradient — not useful as a track.

### Narrow/broad component issue
- Awad+2023 (A&A 683, A14) showed Jhelum has two components
- Narrow: σ_v = 4.8 km/s, [Fe/H] dispersion = 0.15
- Broad: σ_v = 19.5 km/s, [Fe/H] dispersion = 0.34
- Their member list is in Appendix C but NOT in any public data archive
- Cannot reproduce component separation without their catalog
- Current track uses combined members — dominated by narrow component
  in most bins (more stars per unit area)

### Total: 6 phi2 + 6 pm1 + 6 pm2 + 6 rv = **24 data points**

## Orphan-Chenab (86 data points)

### Source catalog
- **Paper:** Koposov et al. 2023, MNRAS
- **Data:** Zenodo 7222654 (FITS files)

### Sky track
- Source: `orphan_M_track_bins.fits` → 19 bins originally
- **Removed phi1 = −90 bin** (phi2_err = 2.79°, 10× worse than all others, no PM data)
- **Result: 18 bins** saved to `data/orphan/orphan_track.csv`

### PM track
- Source: `orphan_pm1_bins.fits`, `orphan_pm2_bins.fits` (Zenodo 7222654)
- 20 bins each, merged into track by phi1 matching
- 17/18 sky bins have matching PM data
- Published errors from FITS percentile intervals
- **Result: 17 pm1 + 17 pm2 bins** (columns in orphan_track.csv)

### RV track
- Source: `orphan_rv_bins.fits` (Zenodo 7222654)
- Separate phi1 grid from sky track
- **Result: 20 bins** saved to `data/orphan/orphan_rv_track.csv`

### Distance track (script: `14_rebuild_derived_tracks.py`)
- Source: `orphan_dmrr_bins.fits` (Zenodo 7222654)
- **Method:** RR Lyrae distance modulus measurements
- **Conversion:** d = 10^((DM+5)/5) / 1000 kpc
- **Error propagation:** d_err = d × ln(10)/5 × edm
- **Result: 14 bins** spanning 16.8–76.7 kpc, errors 0.3–3.0 kpc
- Saved to `data/orphan/orphan_dist_track.csv`

### Unused Koposov+2023 products (not relevant for Omega_p)
- `orphan_M_width_bins.fits` — stream width (7 bins)
- `orphan_M_lindens_bins.fits` — linear density (100 bins)
- `orphan_M_pmwidth_bins.fits` — PM dispersion width (5 bins)
- `orphan_pm1sig_bins.fits` — PM1 dispersion (8 bins)
- `orphan_pm2sig_bins.fits` — PM2 dispersion (8 bins)
- `orphan_rvdisp_bins.fits` — RV dispersion (5 bins)
- `orphan_dmsigrr_bins.fits` — distance modulus scatter (6 bins)
These are density, width, and dispersion products. The Omega_p likelihood
uses stream TRACK POSITION, not density or dispersion.

### Total: 18 phi2 + 17 pm1 + 17 pm2 + 20 rv + 14 dist = **86 data points**

## Rotation Curve (32 data points)

- **Paper:** Eilers et al. 2019, ApJ 871, 120
- **Data:** `data/rotation_curve/eilers2019_rc.csv`
- 32 points from R = 5–21 kpc
- Asymmetric errors symmetrized: σ = (σ+ + σ−) / 2
- Loaded at import time in `src/likelihood/rotation_curve.py`
